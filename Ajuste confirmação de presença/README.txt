O site foi preparado para enviar as confirmações para um backend privado ligado a esta planilha.

O que falta fazer no Google:
1. Abrir a planilha acima.
2. Ir em Extensões > Apps Script.
3. Colar o arquivo backend abaixo no projeto.
4. Salvar e Implantar como aplicativo da Web.
5. Copiar a URL do aplicativo da Web e colar em RSVP_BACKEND_URL no index.html.

Chave de acesso do painel: TROQUE_SUA_CHAVE_AQUI

Observação: a planilha não precisa ser pública. Só você verá os confirmados quando o backend estiver implantado.
